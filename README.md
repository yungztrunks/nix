# nix os config

will expand & improve the readme in the future, for now:

current features:
- somewhat easy to manage multiple hosts / users (with just commands)
- easy alias
- multiple desktop environments as nix specifications (choosable on boot)
  - optional, you can disable build & entry in boot menu too
- flakes & home manager
- noctalia shell included
- looks normal

I still have more features that I want to add including window bars (I want them to look like macOS) and some other features.
my configuration is 100% overkill for a single user on a single laptop but I like to have things just in case.
some things (especially the just commands) have not been tested at all and were just added with AI; will evaluate them later in time
